Typography:


Headings: a confident, slightly condensed sans-serif with real weight
contrast — use Fraunces (a serif with character) for H1/H2 only, to give
the brand a distinct "established, trustworthy" feel instead of looking
like every other SaaS template. Import from Google Fonts.
Body/UI text: Inter for everything else (forms, nav, body copy, buttons) —
clean and highly legible.
Set a real type scale: H1 ~48px/56px line-height, H2 ~32px, H3 ~22px,
body ~16px/1.6, small/caption ~13px. Use consistent font-weight steps
(400 body, 600 emphasis, 700 headings) — don't default everything to bold.
